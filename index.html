I am a front-end developer in Toronto, Ontario. I am proud to be a graduate of HackerYou's Web Development Immersive (Bootcamp) program. Writing and debugging the web in Javascript, HTML and CSS is fun!

I also enjoy learning computer science concepts such as language design and implementation. For example, I built a simplified Python interpreter — in Python!

Every project requires flexibility, but my ideal creative process is guided by principles: Listening before speaking. Whiteboard before keyboard. Consistency before speed. Clarity before cleverness.
