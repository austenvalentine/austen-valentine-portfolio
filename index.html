About Me

Hi, I’m Austen! I’m a front-end developer from Toronto. My creative process is guided by four principles: Listening before speaking. Whiteboard before keyboard. Consistency before speed. Clarity before cleverness.

Among my technical skills are version control and collaboration with Git, pixel perfect styling with SASS and CSS frameworks, dynamic content with JavaScript, jQuery and React.js, NoSQL with Firebase, responsive web design, and web accessibility. I also enjoy learning computer science concepts such as language design and implementation. For example, I built a simplified Python interpreter — in Python!

If you have any questions or would like to say hi, contact me.
