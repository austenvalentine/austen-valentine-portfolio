About Me

Name : Austen Valentine
Profession : Web Developer
Skills : JQuery & React.js
Twitter : @austenfound

I am a front-end developer in Toronto, Ontario. I am proud to be a graduate of HackerYou's Full-Time Web Development Immersive program (aka boot camp). Writing and debugging the web in Javascript, HTML and CSS is fun!

Among my technical skills are version control with Git, compiled stylesheets with SASS, dynamic content with JQuery and React.js, NoSQL with Firebase, responsive web design, and accessibility for screen-readers and keyboard navigation. I also enjoy learning computer science concepts such as language design and implementation. For example, I built a simplified Python interpreter — in Python!

Every project requires flexibility, but my ideal creative process is guided by principles: Listening before speaking. Whiteboard before keyboard. Consistency before speed. Clarity before cleverness.
