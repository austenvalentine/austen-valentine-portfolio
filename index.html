About Me

I am a front-end developer in Toronto, Ontario. I am proud to be a graduate of HackerYou's Full-Time Web Development Immersive program (aka boot camp). Writing and debugging the web in Javascript, HTML and CSS is fun!

Every project requires flexibility, but my ideal creative process is guided by principles: Listening before speaking. Whiteboard before keyboard. Simplicity before sophistication. Consistency before speed. Clarity before cleverness.

Among my technical skills are version control with Git, compiled stylesheets with SASS, prototyping with JQuery, implementation with React.js and Firebase, responsive web design and web accessibility for dynamic content. I also enjoy learning computer science concepts such as language design and implementation. For example, I built a simplified Python interpreter — in Python!.
